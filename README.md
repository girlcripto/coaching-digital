# Coaching Digital

Landing page moderna e responsiva desenvolvida para o lançamento do projeto **Coaching Digital**, focada em performance, autoridade e conversão no ambiente online.

##  Licença

Este projeto está licenciado sob a licença MIT.

## �🚀 Tecnologias Utilizadas

* **HTML5 Semântico:** Estrutura limpa e acessível.
* **CSS3:** Estilização avançada com *Dark Glassmorphism*, gradientes, variáveis CSS e animações suaves (`@keyframes`).
* **JavaScript:** Envio assíncrono do formulário de lead para o servidor local.
* **Node.js:** Servidor HTTP simples para servir os arquivos e receber os leads.
* **Design Responsivo:** Layout adaptável para dispositivos móveis e desktops utilizando CSS Grid e Flexbox.
